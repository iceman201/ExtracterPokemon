# ExtracterPokemon

## Introduction
An LUA program to extract all names of pokemon icon and pokemon in pokemon.html, then rename all the pokemon icon.

##How to use it
* Linux: 
	* ```./rename_all_icon.sh```
* Mac OS: 
	* ```bash rename_all_icon.sh```
